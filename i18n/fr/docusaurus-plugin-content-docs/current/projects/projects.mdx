# Projets

Nos projets servent de base créative et organisationnelle pour le développement
de nos produits. Ils peuvent être basés sur les idées de notre équipe ou sur des
demandes du personnel de l'ACIA.  Les projets sont le cadre stratégique qui nous
permet de créer et d’affiner ces produits, tandis que les produits sont le
produit final de notre innovation, conçu pour l’interaction directe et la
satisfaction des utilisateurs.

Voici les projets sur lesquels nous travaillons actuellement :

- [Invasive Insects Project](./invasiveinsects/introduction.md)
- [Poultry and Red Meat Condemnations
  Project](./poultryredmeatcond/introduction.md)
